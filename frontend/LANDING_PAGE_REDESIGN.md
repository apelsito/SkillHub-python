# SkillHub Landing Page Redesign

## Design Direction

The landing page presents SkillHub as an enterprise skill registry for AI agent teams. It uses a restrained product-app visual language: clear navigation, searchable entry points, concise value propositions, and direct calls to action.

## Key Elements

- Hero section with product positioning, search, and primary actions.
- Feature grid covering private deployment, versioning, search, collaboration, governance, and CLI compatibility.
- Quick-start block with environment setup, install, and publish commands.
- Footer links for documentation, GitHub, community, privacy, and terms.

## Implementation

- React and Vite.
- Tailwind CSS utility styling.
- Canvas-based background particle animation.
- Copyable quick-start commands.
- Vite proxy to the backend at `http://127.0.0.1:8080`.

## Local Preview

```bash
cd frontend
pnpm dev
```

Open `http://127.0.0.1:3000`.
